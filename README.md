#[Luso CMS](http://lusocms.org) 


##Features Overview

- Created with Laravel 3
- Composer
- Themes
- Caching system
- One folder wrapping upgradable core
- Multi location for bundles
- Core and user bundles
- View overriding for bundles
- More to come...

##Quick Install

- Run composer update.
- Got to your url(example.com).
- Follow the setup.
- Enjoy.
